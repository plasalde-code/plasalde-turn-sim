▶ P.Lasalde Turn Simulator

AASHTO vehicle turning path analysis — here's how to get started

✏️

Drawing a Path

  • Click Draw then click on the canvas to place path nodes
  • Place at least 2 nodes to see the vehicle simulation
  • Right-click (or auto-switches) to enter Select mode
  • In aerial mode: drag to pan map, scroll to zoom
👆

Editing Nodes

  • Click Select then click any node to select it
  • Drag a node to reposition it
  • Set a Curve Radius on middle nodes for smooth turns
  • Endpoints (start/end) cannot have a curve radius
🛰️

Aerial & Background

  • Click Aerial to toggle live satellite imagery
  • Click Load PDF to overlay a site plan
  • Use Scale tool to calibrate real-world units
  • Lock the PDF to geo-reference it to the map
🚛

Design Vehicles

  • 9 AASHTO vehicles — Passenger Car to WB-20 (65 ft)
  • Min turning radius enforced per AASHTO Green Book
  • AASHTO speed check: green / yellow / red status
  • Reverse mode with jackknife detection for semitrailers
📥

KML / KMZ Import

  • Click KML In to import reference geometry
  • Lines, polygons and points appear geo-referenced on the map
  • Import from Google Earth or GIS tools
📤

Export

  • PNG — high-res screenshot for reports
  • DXF — AutoCAD R12 format
  • KML — Google Earth (requires Aerial mode)
  • Undo / Clear buttons in the toolbar
